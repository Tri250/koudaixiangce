Core Engine

专业级 修图引擎

Rust + WGSL 构建的全 32 位浮点图像处理管线，配合 WGPU 跨平台渲染，带来即时反馈的流畅编辑体验。每一次拖动滑块，都是 GPU 上的实时计算。

01 / 06
◈

GPU 加速渲染

全 32 位浮点处理管线，WGSL 着色器实时预览所有调整，无延迟反馈。

  • WGPU 跨平台渲染
  • 高分辨率实时预览
  • 多图层蒙版支持
  • ROI 区域渲染优化
02 / 06
◇

非破坏性工作流

所有编辑存储在 .rrdata 附属文件中，原始图像永远不被修改，随时回溯。

  • 完整撤销 / 重做历史
  • 虚拟副本功能
  • 参数完全可逆
  • 批量复制粘贴
03 / 06
◆

全格式 RAW 支持

基于 rawler 引擎，覆盖绝大多数相机品牌的 RAW 格式与常见图像格式。

  • Canon / Nikon / Sony
  • Fuji X-Trans 传感器
  • DNG / TIFF / JPEG
  • WebP / AVIF / JXL

  • 人像精修与面部重塑
  • AI 降噪 (NIND)
  • 语义搜索与自动标签

  • 画笔 / 线性 / 径向蒙版
  • 颜色 / 亮度 / 深度蒙版
  • AI 主体 / 天空 / 前景

Operation Guide

操作指南 与工作流

从导入到导出，完整的修图工作流。每一步都经过精心设计，让摄影师专注于创作本身。

Step 01

导入图像

开启修图之旅的第一步。

  • 点击「打开文件夹」导入照片库
  • 支持文件夹递归浏览
  • RAW / JPEG / PNG / WebP 等多格式
  • Android 端点击「打开图库」
  • 虚拟副本功能无损复制编辑
Step 02

基础调整

打造完美的色调基础。

  • 使用「自动调整」快速获得基础效果
  • 调整曝光、对比度、高光、阴影
  • 调节色温色调营造氛围
  • 使用曲线工具精细控制影调
  • 添加 LUT 快速套用电影感
Step 03

局部精修

蒙版与 AI 让调整更精准。

  • 使用 AI 蒙版一键选择主体 / 天空
  • 画笔蒙版精细绘制调整区域
  • 线性 / 径向渐变蒙版
  • 蒙版加减交叉组合运算
  • 每个蒙版独立调整参数
Step 04

人像精修

AI 赋能的人像美化。

  • 点击「检测人脸」自动识别
  • 瘦脸、大眼、瘦鼻一键调整
  • 磨皮三算法可选
  • 液化工具自由塑形
  • 美型模块调整身材比例
Step 05

导出分享

输出高质量作品。

  • 支持 JPEG / PNG / TIFF / WebP / AVIF
  • 批量导出提高效率
  • 添加水印保护版权
  • 可选择是否保留元数据
  • 自定义文件命名规则
Tips

快捷技巧

提升效率的小技巧。

  • 按 B 键对比原图 / 效果图
  • Ctrl+Z 撤销，Ctrl+Y 重做
  • 右键缩略图复制粘贴调整
  • 保存预设快速复用风格
  • F 键切换全屏编辑

Changelog

v1.7.4 更新亮点

本次更新带来多项功能升级与稳定性提升，全面完善操作链路完整性。

New

人像精修模块全面升级

新增面部重塑（瘦脸、大眼、瘦鼻、嘴型、下颌、额头、下巴、眉毛）、眼神光、笑容管理、三种磨皮算法、肤色统一、美型（大腿、小腿、手臂、腰部、直角肩、天鹅颈、臀部）、头发精修（祛碎发、发色统一、头发柔顺）等 22 项人像功能。

New

创意工具模块

新增 AI 追色、AI 补光、AI 超清、证件照生成、衣物精修、镜头虚化、老照片修复、季节场景、路人消除等 9 大创意功能模块。

Improve

液化工具全新上线

新增独立液化面板，支持推、拉、收缩、膨胀、旋转、重建 6 种画笔模式，支持撤销笔画和重置网格。

Improve

HDR 与色彩空间管理

新增 HDR 高动态范围面板，支持高光恢复、Ultra HDR 导出、色域警告。新增色彩空间管理面板，支持工作 / 输出色彩空间切换、DCP 相机配置文件、软打样功能。

Improve

黑白转换模块

新增独立黑白转换面板，支持通道混合器、多种滤镜预设、分离调色（棕褐 / 硒 / 铜 / 蓝晒 / 金）。

Fix

Android 端适配优化

修复 Android 自适应图标、修复操作链路完整性、优化 57 个子功能模块的端到端调用、修复液化弹窗渲染问题、补全所有翻译键。

Fix

操作链路完整性修复

全面修复 57 个子功能模块的前端到后端操作链路，统一使用 js_adjustments 模式，确保所有功能完整可用、操作链路完整闭环。
